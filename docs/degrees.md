# Справочник учёных степеней

## Получение списка учёных степеней

`GET /v1/degree/get-all` вернёт список учёных степеней.

### Запрос:

Параметры: отсутствуют.

### Ответ:

```json
[
  {
    "id": 1,
    "name": "Кандидат медицинских наук"
  },
  {
    "id": 2,
    "name": "Доктор медицинских наук"
  }
]
```
