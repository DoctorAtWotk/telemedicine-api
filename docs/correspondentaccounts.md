# Корреспондентские счета

## Получение информации о счёте

`GET /v1/correspondentaccount/get/{id}` вернёт информацию о счёте.

### Запрос:

Параметры: 

Имя | Тип | Описание
--- | --- | ---
id | string | уникальный идентификатор счёта

### Ответ:

```json
{
  "id": "b90ffede-7381-470f-ac9f-6ac9a43bad3b",
  "balance": {
    "amount": 13850,
    "currency": {
      "id": "RUB",
      "name": "руб."
    }
  }
}
```
