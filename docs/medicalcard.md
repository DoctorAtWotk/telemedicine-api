# Медицинская карта

## Получение информации о медицинской карте пациента

`GET /v1/medicalcard/get/{id}` вернёт информацию о медицинской карте пациента.

### Запрос

Имя | Тип | Описание
--- | --- | ---
id | string | уникальный идентификатор пациента

### Ответ:

```json
{
  //.. объект медицинская карта
}
```
